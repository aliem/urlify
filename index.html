urlify.js

urlify.js taken from Django's Admin modified to be included as component module. The original file is here, this is a modified version of the same code with some small enhancements.

Install with component

component install aliem/urlify

API

var urlify = require('urlify')

urlify('Foo  bar bAz ');
// 'foo-bar-baz'

The function accepts a second parameter to set the maximum length of the resulting string:

urlify('Foo bar bAz', 5); // 'foo-b'

The library handles different accents for many charsets. Right now it includes:

  • Latin
  • Greek
  • Turkish
  • Russian
  • Ukranian
  • Czech
  • Polish
  • Latvian
